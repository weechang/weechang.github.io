>设计模式一直以来就是一个比较热门的话题，尤其是在java这样的oo语言里面。如今，设计模式已成为架构师和程序员之间交流的必要语言，也是普通程序员成长为高级程序员的必要基础之一。只有掌握设计模式，我们才能更好地掌握OO分析和设计方法，如DDD域驱动设计，并更加熟练地学习和使用新兴语言。

<!-- more -->
